# X Unfollowers Radar (推特未回关雷达) 📡

一个极致轻量、安全、无需抓包的纯前端 X/Twitter 未回关检测脚本。

## 🌟 特点
- **零网络请求**：完全抛弃危险的底层 API 抓包，绝不触发 Twitter 的安全警报或账号风控。
- **纯 DOM 视觉扫描**：通过检测推特原生的“关注了你 (Follows you)”标签来判断是否互关，永远不会因为底层数据结构改变而失效。
- **极速丝滑 (V4.5+)**：采用 `O(1)` 复杂度的虚拟 DOM 回收过滤算法，仅对“未回关”用户渲染高亮，不给浏览器增加任何负担，千人列表丝滑滚动不卡顿。
- **多语言支持**：内置支持 14 种语言的推特界面（英语、简中、繁中、日语、韩语、法语等）。

## 🚀 包含两个版本

本项目包含两个版本的代码，您可以根据自己的喜好选择使用：

### 1. 浏览器控制台版 (推荐)
- **文件**：`x-unfollowers-radar-console.js`
- **优点**：纯净无残留，无需安装任何插件，绝不报错。
- **使用方法**：
  1. 在 Chrome 中打开您的推特 `Following` (正在关注) 列表页面。
  2. 按 `F12` 打开开发者工具。
  3. 切换到 `Console` (控制台) 标签页。
  4. 复制该文件中的所有代码，粘贴到光标处并按回车运行。
  5. 💡 **高阶技巧**：您可以将代码保存在 Chrome 的 `Sources -> Snippets` 中，以后按 `Ctrl + Enter` 一键运行！

### 2. Tampermonkey (油猴) 版
- **文件**：`x-unfollowers-radar.user.js`
- **优点**：自动注入，每次打开页面都会在后台静默运行。
- **使用方法**：
  1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展。
  2. 点击油猴图标 -> **添加新脚本**。
  3. 清空编辑器，将该文件中的代码全部复制并粘贴进去。
  4. `Ctrl + S` 保存。
  5. 刷新推特的 `Following` 列表，开始愉快的扫雷吧！

## 💡 视觉效果
- **未回关**：被红色的虚线框圈出，附带 `💔 未回关` 徽章。
- **互关**：不加任何额外高亮（保持页面极度清爽，只让未回关的人显眼）。
- 右下角附带静默的扫描监控面板。
